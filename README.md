Protean_CBS is a mock service which provides mock data to Financial Information Providers. Protean_CBS doesn't store/provide any real data. 

# STEPS TO USE
- download the `cbs_server.js` under Releases
- execute `node cbs_server.js`
- the server will be started at `port:3000`
- update `cbs.url` in `application.properties` with `http://localhost:3001/api/cbs`

# DEVELOP
- to install modules - run `npm install`
- to start the server - run `npm run dev`
- the server starts @port `3001`
- to build  - run `npm run build`

##### AND THAT'S IT!
